What it does

Serve real local workflows from the device in your hand.

Expose is designed for quick testing, demos, QA passes, and nearby automation where a private same-network endpoint is faster than a cloud setup.

API

Mock JSON, HTML, and text routes

Create hand-written endpoints for browser tests, prototypes, QA fixtures, and local integration work.

DIR

Share files and folders read-only

Pick documents from iOS Files and expose them as local HTTP responses without uploading them elsewhere.

LOG

Watch requests as they happen

Inspect methods, paths, status codes, durations, byte counts, and matched routes during a local session.

Local by design

Useful when you need a server, not an account.
