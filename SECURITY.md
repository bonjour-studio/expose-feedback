# Security Policy

Please do not open public issues or discussions for security vulnerabilities.

Report security concerns privately to:

support@bonjour.dev

Useful reports include:

- Expose version and build number.
- Device model and iOS/iPadOS version.
- A clear description of the behavior.
- Minimal reproduction steps.
- Whether token protection, CORS, host validation, client IP allowlists, file routes, photo routes, or local network access are involved.

Do not include real access tokens, private files, private local network URLs, or personal data unless you are comfortable sharing them by email.
