# Support

## Public Help

Use GitHub Discussions for questions, ideas, and workflow sharing:

- Q&A: https://github.com/bonjour-studio/expose-feedback/discussions/categories/q-a
- Ideas: https://github.com/bonjour-studio/expose-feedback/discussions/categories/ideas
- Show and tell: https://github.com/bonjour-studio/expose-feedback/discussions/categories/show-and-tell

## Bug Reports

Use Issues when you can describe a reproducible problem:

https://github.com/bonjour-studio/expose-feedback/issues/new/choose

Please include the Expose version, device model, iOS/iPadOS version, reproduction steps, and any relevant screenshots with sensitive data removed.

## Private Support

Email support@bonjour.dev for:

- Billing and purchase questions.
- Privacy requests.
- Reports that include private files, tokens, URLs, logs, email addresses, or other personal data.
- Security vulnerabilities.

The in-app Contact Developer button also opens this email address and pre-fills version and device diagnostics. You can edit or remove those details before sending.
